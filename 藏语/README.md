# 藏语直播助手APP

## 项目简介
藏语直播助手是一款专为藏族主播设计的直播辅助工具，旨在帮助藏族主播提升直播质量和带货效果。应用采用抖音式黑白配色主题，界面简洁现代，操作便捷，并融合了多项AI辅助功能，特别针对藏语特色进行了优化设计。

## 核心功能

### 1. 藏语-普通话谐音标注技术
- 支持藏语发音音节拆分
- 提供汉语谐音字和拼音标注
- 实时发音辅助功能

### 2. AI定制化带货话术生成
- 针对藏区特色农产品（虫草、牦牛肉、青稞制品、手工艺品等）自动生成适配话术
- 支持多种话术风格设置（亲切、专业、促销等）
- 场景模板选择和个性化调整功能

### 3. 藏区特色虚拟背景
- 提供高清藏区场景素材库
- 一键切换直播背景功能
- 支持自定义背景上传

### 4. 实时发音辅助与反馈
- 通过三色系统实时反馈发音标准度
- 提供发音改进建议和训练指导

### 5. 简易数据可视化
- 使用藏区特色元素（牦牛、青稞等）直观展示直播数据
- 实时显示观众互动和销售数据

### 6. 多平台账号体系
- 支持抖音、微信账号登录
- 手机号/邮箱注册功能
- 用户信息安全管理

### 7. 论坛交流功能
- 专为藏族主播打造的经验分享社区
- 方言交流专区
- 成功案例分享和问题求助区

## 技术栈

- **前端框架**：纯HTML/CSS/JavaScript（无框架依赖，易于部署）
- **UI组件库**：Tailwind CSS (CDN) + 自定义组件
- **图标库**：Font Awesome (CDN)
- **数据可视化**：Chart.js (CDN)
- **样式设计**：抖音式黑白配色主题
- **存储方案**：本地 localStorage 存储
- **模拟API**：内置模拟数据服务

## 特色设计

### 抖音式黑白配色主题
- 主黑色 (#000000)：背景色
- 次黑色 (#121212)：卡片、导航栏等组件背景
- 三级黑色 (#1e1e1e)：输入框、标签等元素背景
- 四级黑色 (#252525)：悬停状态背景
- 主白色 (#ffffff)：主要文本颜色
- 次白色/灰色 (#aaaaaa)：次要文本颜色
- 抖音红 (#ff0050)：强调色和交互元素

### 藏族文化元素融入
- 数据可视化中使用藏族特色符号（牦牛、青稞、雪山等）
- 话术模板中融入藏族文化元素
- 虚拟背景库提供藏区特色场景

### 响应式设计
- 支持各种移动设备屏幕尺寸
- 针对直播场景优化的界面布局
- 符合移动端操作习惯的交互设计

## 使用技巧

1. **直播前准备**
   - 使用AI定制话术生成功能提前准备直播脚本
   - 设置合适的虚拟背景增强直播效果
   - 熟悉谐音标注功能以提高与观众的沟通效率

2. **直播过程中**
   - 实时关注发音反馈系统，改进藏语发音
   - 利用数据可视化面板了解观众互动情况
   - 适时切换话术风格，保持直播氛围

3. **直播后优化**
   - 在个人主页查看直播效果分析
   - 在论坛分享直播经验，获取反馈
   - 保存成功话术到素材库，便于后续使用
